🖼️ YOLO-Django-Project

Real-time Object Detection using Django & YOLOv8

🎯 Goal of the Project

  • Upload an image and run YOLOv8 detection
  • Display image with bounding boxes & labels
  • Show list of detected objects with confidence scores

💡 How It Works — Step by Step

  1. Open the web page and upload an image.
  2. Click Detect.
  3. Django saves the uploaded image.
  4. YOLOv8 processes the image.
  5. Bounding boxes & labels are drawn.
  6. Results page shows the processed image & detected objects list.

📂 Project Structure (Key Files)

⚙️ Installation & Setup

  1. Clone the repository:
    git clone https://github.com/IqramZargar/WEB-APPLICATION-For-Object-Detection.git
  2. Create virtual environment:
    python3 -m venv venv
    source venv/bin/activate
  3. Install dependencies:
    pip install -r requirements.txt
  4. Download YOLOv8 weights:
    wget https://github.com/ultralytics/assets/releases/download/v0.0.0/yolov8n.pt
  5. Apply migrations:
    python manage.py migrate
  6. Run server:
    python manage.py runserver

🛠 Tech Stack

  • Backend: Django
  • ML Model: YOLOv8 (Ultralytics)
  • Language: Python
  • Frontend: HTML + Django Templates
  • Image Processing: OpenCV
